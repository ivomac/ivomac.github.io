Settings

Description
  • Todo list backed by your own Turso (SQLite) database.
  • Access and edit your todos from anywhere.
  • Warning: Credentials are stored in localStorage only — no cookies or other storage. "Clear Credentials" removes them entirely.
How to Setup
  1. Create a free account at turso.tech
  2. Create a database:
    • Via CLI: turso db create my-todos
    • Or via the Turso web dashboard
  3. Get the DB URL:
    • Via CLI: turso db show my-todos --url
    • Looks like libsql://my-todos-org.turso.io
  4. Create an auth token:
    • Via CLI: turso db tokens create my-todos
    • Or via the Turso web dashboard
  5. Enter the URL and token below and save
How to Use
  • Add items: Type in the input field and press Enter or click +
  • Mark done: Hover over an item and click ✓
  • Copy items: Hover over an item and click 📋
  • Refresh: Click the 🔄 icon to reload from Turso

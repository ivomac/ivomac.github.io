Todo List

Description
  • Simple todo list application that synchronizes through GitHub Gists.
  • Access and edit your todos from anywhere.
  • Each todo item is a line in a plain text file.
  • Warning: Secrets are stored locally but are not encrypted.
How to Setup
  1. Create a (secret) GitHub gist
  2. Note the gist ID from the URL and the filename (e.g., "todo")
  3. Generate a GitHub Personal Access Token:
    • Go to github.com/settings/tokens
    • Click "Generate new token"
    • Select ONLY the "Gists" scope
    • Save the generated token
  4. Enter all three values below and save
How to Use
  • Add items: Type in the input field and press Enter or click +
  • Delete items: Hover over an item and click the trash icon
